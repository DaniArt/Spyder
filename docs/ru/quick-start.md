# Быстрый старт

Минимальный сценарий: запуск → захват → копирование селектора → сохранение локатора.

## 1) Запуск

Запусти `Spyder.exe`.

## 2) (Опционально) VCL интроспекция

Если нужна работа на уровне VCL компонентов:
1. Включи VCL в UI.
2. Выбери процесс цели.
3. Дождись статуса подключенного VCL.

Без VCL Spyder продолжит работу через Win32/UIA/MSAA.

## 3) Подсветка при наведении

Наведи курсор на элемент:
- Spyder нарисует подсветку.
- Панели обновятся снимком метаданных.

## 4) Захват

Используй:
- хоткей (если настроен),
- или drag-capture.

После захвата обновятся:
- иерархия,
- свойства,
- селектор.

## 5) Копирование селектора

Нажми Copy рядом с полем селектора.

## 6) Сохранение в `locator.json`

1. Открой менеджер локаторов.
2. Выбери путь к `locator.json`.
3. Добавь/обнови запись текущим захватом.

## Логи

`%LOCALAPPDATA%\Spyder\logs\spyder.log`
